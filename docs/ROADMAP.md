# ROADMAP — FC Online Lab

> 2026-07 기획 확정본. 조사 근거는 하단 "리서치 요약" 참조.

## 비전

FC온라인 커뮤니티의 **반복 질문을 자동화**하고, **논쟁을 데이터로 판정**하고,
**유저 대회를 API로 자동 검증**하는 올인원 서비스.
기존 서비스(조회·나열형)와 달리 "진단·자동화·검증"에 집중한다.

## 사이트맵 (목표 상태)

```
/                          홈 — 검색 + 오늘의 VS + 인기 스쿼드
/user/[nickname]           전적 조회 (모든 기능의 토대)
/squad-clinic              ① AI 스쿼드 평가 + 예산 스쿼드 생성기
/vs                        ② VS 판독기 (선수 비교 + 투표)
/real-squad                ③ 현실 라인업 스쿼드 생성기
/community
  /clubs                   ④ 클럽 모집 게시판
  /tournaments             ⑤ 유저 대회 (API 자동 검증 대진표)
  /friendlies              ⑥ 클럽 교류전
/profile                   내 프로필 (ouid 연동, 내 스쿼드, 참가 대회)
```

## 단계별 로드맵

### Phase 0 — 스캐폴드 + 기획 문서 ✅ (현재)
Next.js 16 + Tailwind v4 스캐폴드, 기획/API/스키마 문서.

### Phase 1 — 넥슨 API 코어 (모든 기능의 토대)
- `lib/nexon/` 클라이언트: id → ouid → basic/maxdivision/match/match-detail
- 메타데이터 로더 + 캐시 (matchtype/spid/seasonid/spposition/division)
- 선수 이미지 프록시 라우트 (액션샷 → 기본 이미지 → 실루엣 폴백)
- 레이트리밋 순차 큐 + match-detail 영구 캐시 (Supabase)
- `/user/[nickname]` 전적 페이지 (등급, 최근 경기, 기본 통계)

### Phase 2 — 스쿼드 클리닉 (핵심 차별화 MVP)
- 최근 경기에서 사용 스쿼드 자동 추출 (match-detail의 player[])
- 평가 엔진: 급여 효율·포지션 적합도·랭커 대비 스탯(ranker-stats)
- Claude 리포트: 점수 + 약한 고리 + 교체 추천 (BlogLab 진단 패턴 이식)
- 결과 누적 저장 → 변동 추적

### Phase 3 — VS 판독기 (바이럴 엔진)
- 선수 2명 검색 → ranker-stats 실사용 평균 비교 카드
- 유저 투표 오버레이 + OG 공유 이미지 (Vercel OG)

### Phase 4 — 현실 라인업 스쿼드
- API-Football 연동 (최근 경기 라인업 + 포메이션 grid)
- 실선수 ↔ fc_pid 매핑 테이블 (Claude 배치 매칭 + 수동 확인 UI, 영구 축적)
- 시즌 카드 선택 정책: 최신 시즌 → 랭커 사용 카드 폴백
- 피치 렌더 + 공유 이미지

### Phase 5 — 커뮤니티 (트래픽 확보 후)
- 5a. 클럽 모집: 게시판 + 지원자 전적 카드 자동 첨부 (BlogLab 서이추 패턴)
- 5b. 유저 대회: 8~16인 무상금, ouid 등록 → 매치 API 결과 자동 검증 대진표
  - 선행 검증 필요: 1:1 친선(초대) 매치의 matchtype·반영 속도
- 5c. 클럽 교류전: 대회 엔진 재사용 (일정 + 멤버별 매치 검증 + 팀 스코어 집계)

## 원칙

- **순서가 전략**: 도구(1~4)로 유입 → 커뮤니티(5)로 리텐션. 커뮤니티 선출시 금지(콜드스타트).
- **시세 기능 금지(v1)**: 공식 API에 가격 없음 + 시세조작·넥슨 개입설 등 민감 영역.
- **상금 대회 금지(v1)**: 경품·사행성 규정 리스크. 명예 랭킹/뱃지로 대체.
- 매핑 테이블(실선수↔카드)과 match-detail 캐시는 쌓일수록 독점 자산 — 처음부터 스키마에 반영.

## 리서치 요약 (2026-07 조사)

- **커뮤니티 반복 주제**: 시세 불신(넥슨 개입설·프티 사건), 강화 확률, "스쿼드 평가 좀"/"X억으로 뭐 사요"(반복 질문형), 메타·패치 논쟁, X vs Y 선수 비교, 어뷰징.
- **경쟁 서비스**: 피파어딕트·피온디비·FC INFO(선수 DB/시세), FC INNO(전적), 공식+FC-RANK(수동 스쿼드메이커). 자동 평가/생성·데이터 판정·API 검증 대회는 전무.
- **유저 대회 현황**: 공식(FSL)·범용 플랫폼(LVUP, 배틀리카)은 있으나 소규모 유저 대회는 디스코드+스크린샷 인증 수동 운영.
- **라인업 데이터**: API-Football 무료 플랜(100req/일)이 라인업+포메이션+grid 제공. football-data.org 무료 티어는 라인업 없음.
- FC온라인 GitHub 생태계는 토이 수준(최고 14★) — 가져다 쓸 완성품 없음, 참고용: bluuewhale/nexon-openapi-python(스키마), hskimim/fconline-coach-agent(AI 코치 아이디어).
