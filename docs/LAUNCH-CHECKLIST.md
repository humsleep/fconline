## 3.5) 🔴 /meta 픽 랭킹 시딩 (오픈 전 필수 — 회의 P0 1순위)

픽 랭킹은 전적 검색 데이터(match_cache)에서 만들어지므로 오픈 첫날은 확정적으로 빈 화면.
오픈 전에 아래 순서로 시딩:

1. 프로덕션 사이트에서 **랭커/지인 구단주명 20~30건 검색** (공식 랭킹 상위권 닉네임 활용) → match_cache 축적
2. 크론 수동 1회 실행:
   `curl -H "Authorization: Bearer $CRON_SECRET" https://<도메인>/api/cron/ranker-snapshot`
3. `/meta` 접속 → 공격/미드/수비/GK 4개 라인이 채워졌는지 확인 (1시간 캐시라 최대 1시간 후 반영)
4. 이후는 자동: 크론이 매일 갱신하고, 검색이 없어도 직전 스냅샷 조합을 재예열(폴백 구현됨)

시딩이 어려우면 오픈 시 /meta를 내비에서 임시 숨김(홈 카드 + 헤더 + 모바일 탭 3곳).
