# FUN ROADMAP — "매일 오고, 자랑하고, 겨루게" (2026-07 기획)

> 원칙: 게임 밖에서도 FC온라인이 재미있어지게. 모든 항목은 현재 보유 데이터로 구현 가능 여부를 표기.
> 노력: S(반나절) / M(1~3일) / L(1주+)

## 루프 1 — 데일리 훅 (내일 또 올 이유)

| 항목 | 내용 | 노력 | 데이터 |
|---|---|---|---|
| ✅ 픽 랭킹 변동 배지 | 전일 대비 ▲▼/NEW — "오늘 뭐가 떴지?" (구현 완료) | S | 보유 |
| 오늘의 내 폼 | 전적 페이지 상단에 최근 5경기 폼(WWDLW) + 직전 대비 평점 변화 화살표 | S | 보유 |
| 이번 주 뜨는 카드 | delta 상위 3장을 홈에 티저로 — 픽 랭킹 유입 훅 | S | 보유 |
| 연속 방문 스트릭 | "🔥 3일 연속 방문" localStorage 배지 (계정 불필요) | S | — |

## 루프 2 — 자랑 (공유가 곧 마케팅)

| 항목 | 내용 | 노력 | 데이터 |
|---|---|---|---|
| 주간 스쿼드 챌린지 | "이번 주: 24TOTS로만 11명" — 대회 유형 글로 운영자 발제, 스쿼드 첨부로 참여, 베스트는 다음 주 고정 | 운영 | 보유 |
| 명예의 전당 | 챌린지/대회 우승 스쿼드 고정 게시판 (커뮤니티 상단 고정) | S | 보유 |
| 진단 등급 도장 공유 강화 | 공유 카드에 클리닉 밴드(최상위/재정비…) 도장을 크게 — "나 상위권임" 인증 문화 | S | 보유 |

## 루프 3 — 겨루기 (커뮤니티 게임화)

| 항목 | 내용 | 노력 | 데이터 |
|---|---|---|---|
| 베스트 답변 채택 | 평가 요청 글에서 작성자가 댓글 채택 → 채택 수가 프로필에 "⚽ 코치 포인트"로 누적 | M | 스키마 +1열 |
| 코치 랭킹 | 코치 포인트 월간 랭킹 페이지 — 잘 아는 유저가 대접받는 구조 | M | 위 선행 |
| 픽 랭킹 예측 | "다음 주 ST 1위 맞히기" 투표 → 적중 배지 | M | 보유 |
| 유저 대회 자동 검증 | 대진표 + 매치 API 결과 자동 판정 (로드맵 5b) | L | 실검증 필요 |

## 루프 4 — 탐험 (데이터 가지고 놀기)

| 항목 | 내용 | 노력 | 데이터 |
|---|---|---|---|
| 카드 1:1 비교 | 성적표에서 두 카드 랭커 실사용 스탯 나란히 | M | 보유 |
| 시즌 몰아보기 | 픽 랭킹을 시즌별 필터로 — "아이콘만 보기" | S | 보유 |
| 내 최고의 한 경기 | 최근 30경기 중 최고 평점 경기를 "명경기" 카드로 | S | 보유 |

## 착수 추천 순서

1. **(완료)** 픽 랭킹 변동 배지
2. **오늘의 내 폼 + 이번 주 뜨는 카드** — 데일리 루프 완성 (S+S)
3. **주간 챌린지 1회 운영** — 코드 없이 커뮤니티 활성 실험
4. **베스트 답변 채택 + 코치 포인트** — 커뮤니티 정착의 핵심 인센티브 (M)
5. 반응 보고 예측 게임 / 카드 비교 순
