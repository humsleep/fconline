# 플레이스타일 분석 스펙 (회의 합의, 2026-07)

전문가 4인(데이터 사이언티스트·게임 분석가·UX·헤비 게이머) 제안→비평→합의.

## 방법
룰베이스가 정답. 단 "(등급밴드 × 컨트롤러) 백분위 정규화 + 소표본 empirical-Bayes 수축"을 얹은 결정론 룰베이스.
순수 절대 임계값은 등급 인플레(상위 등급 볼륨 전반↑)로 전원 공격형 쏠림 → 백분위가 흡수.
**콜드스타트**: 코호트 데이터 없을 땐 tier별 고정 anchor + 절대 임계 폴백 + "베타" 배지 →
데이터 누적되면 배치로 경험적 백분위 테이블 갱신(이 배치가 심장). 클러스터링은 오프라인 검증용만.

## 5축 (계산식)
- **A 전개 템포** (0 점유·지공 ↔ 100 직선·역습): possession(백분위 역) + shortShare + directShare
- **B 침투성** (높을수록 스루 침투): penShare(through+lobbed / passTry) 백분위 × penEff 보정
- **C 개인기 의존** (0 조직 ↔ 100 개인 돌파): dribble/G 백분위 (컨트롤러 분리 필수)
- **D 슈팅 성향** (0 박스 선택형 ↔ 100 외곽 다작): effShoot/G 백분위 + boxRatio(inPenalty)
- **E 수비 적극성** (0 관망 ↔ 100 고강도): (tackleTry+blockTry)/G 백분위 + 태클비중

비율 지표(패스 구성비·boxRatio) = 절대 앵커 + tier 미세보정. 볼륨 지표 = 코호트 백분위.
실력 지표(conv·평점·승패·tackleSuccess·foul)는 **축·아키타입에 미투입** → 강점/취약 칩에만.

## 아키타입 10종 (결정리스트, 첫 매칭)
0 데이터부족(보류, G<5) · 1 텐백 카운터 · 2 압박 사냥꾼 · 3 스루 침투 창조자 ·
4 개인기 드리블러 · 5 박스 포처 · 6 외곽 난사 슈터 · 7 실리 역습가 ·
8 티키타카 지휘관 · 9 밸런스 올라운더(폴백). 명명 = base축 + modifier 형용사.

## 강점/취약 칩 (실행품질 오버레이)
강점(초록↑ 최대3): penEff·conv·tackleSuccess·boxRatio·passAcc 상위
취약(빨강↓ 최대2): conv 하위·foul 상위·penEff 하위·offside 상위·점유부족
성향(주황 단색) ≠ 실력(초록/빨강) 색 분리. 근거 수치 툴팁 필수.

## 시각화
① 아키타입 카드(공유 단위, 유형명+태그라인+등급·컨트롤러·신뢰도 배지)
② 양극 슬라이더(빌드업●—롱볼) + 단극 게이지(침투·개인기), 등급 평균 고스트 마커
③ 강점/취약 칩 + 슛 히트맵. 레이더는 "자세히"에 접어서만.

## 엣지
G<5 보류 / 5~15 저신뢰 흐림 / 축별 게이트(passTry<150→A·B, effShoot<25→D 회색) /
empirical-Bayes 수축 K≈10, 비율은 Wilson 하한 / 컨트롤러 분리 백분위 /
등급 초기 3~4밴드 / winsorize p1·p99 / 몰수·초단시간 경기 제외 / GK·수비형은 공격 아키타입 금지.

## 출시 전 P0
(a) 실데이터 5축 상관행렬 r>0.7 병합 (b) 손설계 경계 vs 오프라인 밀도 봉우리 1회 검증.

## 구현 단계 (콜드스타트 우선)
1. **v1 (지금)**: tier별 고정 anchor + 절대 임계 폴백, 5축 계산, 아키타입 결정리스트, 강점/취약, 양극 슬라이더 + 카드 + 히트맵. "베타" 배지. G 게이트 + 수축(코호트평균=anchor).
2. **v2 (데이터 후)**: match_cache 누적 → (tier×ctrl) 경험적 백분위 테이블 배치 생성 → 백분위 정규화로 승급.
